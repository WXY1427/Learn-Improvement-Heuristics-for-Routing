# Learn-Improvement-Heuristics-for-Routing

A deep reinforcement learning framework to learn the improvement heuristics for routing problems.

## Running the program

For each problem and size:

### Training

```
python3 run.py
```
or run jupyter file
```
RUN MY MODEL.ipynb
```

### Testing

```
python3 test.py
```
or run jupyter file
```
Test with model.ipynb
```
